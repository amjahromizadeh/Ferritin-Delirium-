\subsubsection{\texorpdfstring{\textbf{Two-sample
MR}}{Two-sample MR}}\label{two-sample-mr}

Integrated MR interpretation. The primary IVW (random-effects) estimate
is non-significant (OR = 1.086, 95\% CI 0.935--1.261; p = 0.282). There
is evidence of heterogeneity (IVW Cochran's Q p = 0.015), but no
evidence of directional pleiotropy (MR-Egger intercept p = 0.506).
Steiger filtering supports the direction Ferritin → Delirium (R²x
\textgreater{} R²y; p = 0e+00). MR-PRESSO detects outliers (global p =
0.017), but the distortion test is non-significant (p = 0.705), so the
outlier-corrected nominal association should be treated as exploratory.

\subsubsection{\texorpdfstring{\textbf{SMR/HEIDI}}{SMR/HEIDI}}\label{smrheidi}

Blood, Brain, Pituitary, Liver, Spleen, KidneyCortex; omics:
eQTL/mQTL/sQTL/pQTL), Ferritin shows a broad cis-mediated architecture,
whereas Delirium retains sparse signals concentrated in a few panels.
Under strict Bonferroni+HEIDI (panel-wise Bonferroni + pHEIDI ≥ 0.01 +
nSNP,HEIDI ≥ 3), Ferritin retains 344 probe--panel records (255 unique
probes; 126 genes), Delirium retains 4 probe--panel records (4 unique
probes; 3 genes), and the strict cross-trait intersection is APOE.
